# Pac-Man-Game-By-Python-Turtle
🎮 Pac-Man Game (Python Turtle) A classic Pac-Man-style game developed in Python using the Turtle graphics library as part of a Computer Graphics course project. The game recreates the core mechanics of the original Pac-Man, including player movement, enemies (ghosts), pellets, power-ups, and a scoring system.

# 💡 Features:

Player-controlled Pac-Man with smooth keyboard movement (W/A/S/D)
Multiple enemies with basic AI that chase the player
Collectible pellets and power pellets
Score and lives tracking system
Sound effects for actions (eat, death, start)
Win and Game Over conditions
Dynamic speed boost when collecting power pellets

# 🧠 Game Mechanics:

Enemies actively follow the player using simple chasing logic
Collision detection with walls, enemies, and collectibles
Random spawning system for player and enemies
Temporary power-ups that affect gameplay

# 🛠️ Technologies Used:

Python
Turtle Graphics
winsound for sound effects

# 📚 Project Purpose:
This project was developed for a Computer Graphics course to demonstrate concepts such as:

2D rendering using Turtle
Event handling (keyboard input)
Game loops and real-time updates
Object-oriented programming (OOP)
Collision detection

# 🚀 How to Run:

Make sure Python is installed
Place all required assets (images & sounds) in the correct paths
Run the main file:
```
python main.py
```
